Errors

coldgui.errors.MultipleInstanceError

Raised when something which should be created once in a single process have instances more than one.

coldgui.errors.UnknownAlignError

Raised when user enters an unknown text aligning format format. See aligning preferences. Valid formats are: center, right, left, top, bottom, top-right, top-left, bottom-right, bottom-left

Widgets

A widget is simply an interactable component to control various events. They may not be interactable always, there are different types of widgets to fulfil any task. Also there are a few themes for the widgets, win95 style being the default one.

MainWindow

Main window.

Label

class coldgui.Label(parent, **fields)

A single-line text surface. parent positional argument must be entered.

Argument Default Description
position (0, 0) Widget's position relative to it's parent
text "" Widget's text content
sysfont Calibri System font used to render the text
font None External font file used to render the text
font_size 10 Size of the font
fore_color (0, 0, 0) Color of the text in RGB format
back_color None Back color of the text in RGB format
antialias True Antialiasing option of the text
active True State of the widget
visible True Visibility of the widget
tag None Special tag for the widget. Can be removed in further versions!

Button

class coldgui.Button(parent, **fields)

A clickable button. parent positional argument must be entered.

Argument Default Description
position (0, 0) Widget's position relative to it's parent
width 0 Forces widget to size itself to specific width, if 0, best size is calculated.
height 0 Forces widget to size itself to specific height, if 0, best size is calculated.
padding 0 Space between widget's border and widget's content
text "" Widget's text content
text_align center Position of the text within the widget's borders. See aligning preferences.
sysfont Calibri System font used to render the text
font None External font file used to render the text
font_size 10 Size of the font
border_size 2 Size of the widget's borders
style win95 Theme of the widget. See styling preferences.
active True State of the widget
visible True Visibility of the widget
tag None Special tag for the widget. Can be removed in further versions!

There are some attributes which can be changed after the initialization of the widget's instance:

Attribute Description
font_fore_color Widget's text's color
body_color Widget's body (background) color
