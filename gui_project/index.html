GUI Project


GUI Project is a graphical user interface toolkit powered by Pygame

Altough GUI Project uses Pygame, the main purpose of it is not being a library to used while making games.
It provides various themeable widgets and a simple interface to let users achieve their goals easier.



You can see a cool little example in the right ;)
You can also find more examples in the Examples section.

Introduction

A simple GUI Project project's work flow should be like this:
importing GUI Project > creating the top level window > adding widgets > starting the main loop

from gui import * #Import GUI Project to your project

mw = MainWindow((800, 600), title="My project!") #Create your top level window

mw.start() #Start the main loop

Basics of Widgets

A widget is simply an interactable component to control various events.
They may not be inceractable always, there are different types of widgets to fulfil any task.
